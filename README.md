# 🌱 Agro Forte - Agrinho 2026

## Tema

**Agrinho 2026: Tecnologia, Sustentabilidade e Inovação no Campo**

## Sobre o Projeto

O projeto **Agro Forte** foi desenvolvido com o objetivo de demonstrar a importância da agricultura sustentável e do uso da tecnologia no campo para garantir a produção de alimentos, a preservação dos recursos naturais e a qualidade de vida das futuras gerações.

Através de um site moderno e interativo, o projeto apresenta informações sobre:

* Agricultura sustentável
* Preservação ambiental
* Uso consciente da água
* Energias renováveis
* Agricultura de precisão
* Inteligência Artificial no campo
* Inovação tecnológica
* Produção responsável

## Objetivos

* Incentivar a conscientização ambiental.
* Demonstrar a importância do agronegócio para a sociedade.
* Mostrar como a tecnologia contribui para o desenvolvimento sustentável.
* Valorizar o trabalho dos produtores rurais.
* Promover a educação e a inovação no campo.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Visual Studio Code
